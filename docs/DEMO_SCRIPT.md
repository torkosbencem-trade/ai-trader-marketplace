# Demo Script

Recommended presentation order:

1. Open /demo
2. Open /strategy-builder and upload CSV evidence
3. Open /admin and approve strategy
4. Open /marketplace and show strategy report
5. Submit allocation request
6. Open /allocation-requests and approve request
7. Open /execution and prepare deployment package
8. Open /portfolio and show saved deployment
9. Open /system/audit and show audit trail
10. Open /readiness and explain production status

Demo boundary:

This is an MVP prototype. It does not execute real trades, manage real capital, or connect to live broker accounts.
